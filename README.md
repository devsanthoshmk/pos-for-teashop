# Setting Up

## Clone this repo

``` bash
git clone git@github.com:devsanthoshmk/pos-for-teashop.git
cd pos-for-teashop

```

## Create virtual Envronment

``` bash
python -m venv venv
```

## Activate venv
#### Command Prompt
``` cmd
.\\Scripts\\activate.bat
```
#### PowerShell
``` PowerShell
.\\Scripts\\Activate.ps1
```
#### Bash
``` bash
source venv/bin/activate
```

## Installing Libraries

``` bash
pip install -r requirements.txt
```

# Usage Guide
``` bash
python main.py
```
